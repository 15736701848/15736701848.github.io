GitHub Pages 部署指南

  1. 在GitHub上创建一个新仓库，命名为 yourusername.github.io
  2. 将本页HTML文件保存为 index.html 并上传到仓库
  3. 进入仓库设置 > Pages > 选择部署分支（通常为main/master）
  4. 保存设置，等待部署完成（通常需要1-2分钟）
  5. 访问 https://yourusername.github.io 查看您的博客

GitHub Pages 部署注意事项：

// 确保文件名为index.html
// 避免使用绝对路径引用本地资源
// GitHub Pages不支持服务端脚本
// 使用localStorage存储数据
// API调用需要处理CORS问题
