部署步骤

创建仓库

在GitHub上创建新仓库，命名为 username.github.io（username为您的GitHub用户名）

# 在本地初始化仓库
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/username/username.github.io
git push -u origin main

添加网站内容

创建index.html文件作为网站首页，添加HTML/CSS/JavaScript等静态资源

<!DOCTYPE html>
<html>
<head>
  <title>我的网站</title>
</head>
<body>
  <h1>欢迎访问我的GitHub Pages网站！</h1>
</body>
</html>

启用GitHub Pages

在仓库设置中找到"Pages"选项，选择部署分支（如main）和目录（如/root）

# 部署完成后访问：
https://username.github.io

自定义域名（可选）

添加CNAME文件并配置DNS，将您的域名指向GitHub Pages

# 创建CNAME文件，内容为您的域名：
example.com

# DNS配置添加记录：
TYPE: CNAME
NAME: www
VALUE: username.github.io
